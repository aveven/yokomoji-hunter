## アプリの目的
AIやマーケティングで出てくる横文字（専門用語）に遭遇した時、即座に登録・学習できるアプリ。
「発見の喜び」「覚えた報酬」をゲーミフィケーションで醸成し、自分の成長を実感できる仕様。

## 用語データの型定義

```typescript
export interface Term {
  key: string;        // 用語キー（例: "LLM"）
  fullName: string;   // 英語正式名
  reading: string;    // よみがな
  cat: 'AI' | 'マーケティング' | 'ビジネス' | 'IT' | 'その他';
  rank: 'S' | 'A' | 'B' | 'C';  // 重要度
  impact: number;     // 実務インパクト 0〜100
  freq: '高' | '中' | '低';     // 遭遇率
  explanation: string; // 中学生でもわかる解説
  example: string;    // 使用例
  value: string;      // 覚えるとどうなる？（補助メッセージ）
}
```

## コア機能（設計済み・未実装）

| 画面 | 内容 |
|---|---|
| 発見（Discover） | 用語を入力 → KBから自動補完 → カード表示 |

### 用語カードの表示項目
- 重要度ランク（S/A/B/C）
- 実務インパクト（0〜100点、バーアニメーション）
- 遭遇率（高/中/低）
- 中学生でもわかる解説
- 使用例
- 「覚えるとどうなる？」メッセージ
- 獲得XP

### ゲーミフィケーション要素
- XP（経験値）・レベル
- ストリーク（連続学習日数）
- レベルアップ演出
- 用語を見つけた時のワクワク感・報酬感
